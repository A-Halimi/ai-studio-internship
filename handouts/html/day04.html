2 Field guide the concepts

A neuron is a bouncer at a door. Each input (a reason to open) has a weight (how much it matters). The bouncer multiplies each input by its weight, sums them, adds a personal bias, and runs the total through an activation — the sigmoid squash that turns any number into a 0-to-1 confidence. One neuron draws one straight cut, so it can't split two tangled moons; stack a hidden layer of them and the boundary can bend.

Learning = walking downhill. A loss scores how wrong the guesses are (one number). The gradient says which way each weight should move to shrink it; today you estimate it by nudging each weight and re-checking the loss, then step every weight a little downhill (the learning rate sets the step size). Remember the big secret: everything the network does is matrix multiplication — Weeks 3–4 make those fast.

3 Lab missions check each off in the notebook

  1. ☐Implement the sigmoid 1 / (1 + np.exp(-z)), working on a whole array at once. Expect: sigmoid(0) = 0.5 and a smooth S-curve from 0 to 1.
  2. ☐Matrix multiply, by hand vs NumPy Write the triple-loop matmul_by_hand, then race it against @. Expect: the exact same answer — and NumPy winning by hundreds of times.
  3. ☐Write the forward pass Chain two layers: H = sigmoid(X@W1+b1), then the output layer. Expect: a (300, 1) array of mush near 0.5 — it hasn't learned yet.
  4. ☐Implement the loss Mean squared error: the average of (pred − truth)². Expect: an untrained loss near 0.25.
  5. ☐Take the step (this is learning) Nudge every weight downhill, in place: p -= lr * g. Expect: the loss falls epoch by epoch as the net figures out the moons.
  6. ☐Score it Threshold predictions at 0.5 and take the fraction correct. Expect: a well-trained net lands around 88% on the moons (the ceiling is ~97% — tomorrow's power tools close the gap).
  7. ☐Retrain with 2 and 64 hidden neurons Train both and plot their boundaries side by side. Expect: 2 is too stiff to bend; 64 is far wigglier.

6 Mini-glossary

neuron
Weighted sum of inputs + bias, run through an activation.
weight
How much one input counts — a knob the network tunes.
bias
A neuron's offset — its baseline mood, added before the squash.
activation
The squashing step (sigmoid) that bends straight lines into curves.
hidden layer
Neurons between input and output; more = a wigglier boundary.
loss
One number saying how wrong the predictions are (here, MSE).
gradient
The slope of the loss — which way to move each weight to improve.
learning rate
How big a step to take downhill each update.
